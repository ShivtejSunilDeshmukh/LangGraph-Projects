# LangGraph-Projects 🚀

This repository is part of my journey exploring **LangGraph** and building practical, hands-on use cases. The goal is to understand how graph-based workflows can be used to design intelligent, modular, and scalable LLM applications.

---

## 📌 Projects Included

### 🏏 Cricket Stats Analyzer
A utility to calculate and analyze cricket performance metrics such as:
- Strike Rate   
- Boundaries per Balls  
- Boundary Percentage 

**Use Case:** Helps in understanding player performance using simple statistical calculations.

---

### 👋 Hello World (LangGraph Basics)
A minimal example demonstrating:
- Basic LangGraph setup  
- Simple node and edge flow  
- Execution of a basic graph  

**Use Case:** Ideal starting point for beginners to understand how LangGraph works.

---

### ⚖️ BMI Calculator
A simple health utility that:
- Calculates Body Mass Index (BMI)  
- Categorizes results (Underweight, Normal, Overweight, Obese)  

**Use Case:** Demonstrates input processing and conditional logic using LangGraph.

---

### ✍️ Blog Creator (Prompt Chaining)
A content generation tool that:
- Uses prompt chaining  
- Generates structured blog posts  
- Breaks writing into multiple steps (idea → outline → full blog)  

**Use Case:** Shows how LangGraph can be used for multi-step LLM workflows.

---

### 🧠 UPSC Essay Analyzer
An advanced use-case that:
- Evaluates essays at UPSC level  
- Provides scoring and feedback  
- Analyzes structure, clarity, and depth  

**Use Case:** Demonstrates evaluation pipelines and intelligent analysis using LLMs.

---
